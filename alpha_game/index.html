strong vs b

So, we can make text bold using both the b and strong tags.
But which one should we use?

The strong element is for content that is of "strong importance,"
such as warnings and alerts.

The b element is for content that we want to stand out in the sentence.

The strong tag is used to make it semantically important, while b is
simply styling the text bold.
